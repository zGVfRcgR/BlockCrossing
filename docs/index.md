## 🎮 このプロジェクトについて

**BlockCrossing** は、Minecraft風ブロック世界と踏切の情景を融合させた創作テーマです。  

- ブロック状の遮断機や信号
- 感情を呼び起こす踏切音
- 線路の向こうに広がる想像の物語
